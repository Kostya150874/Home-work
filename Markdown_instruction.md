## Списки

Чтобы добавить ненумерованные списки, надо пункты выделить звездочкой (*). Пример
* Элемент 1
* Элемент 2
* Элемент 3
* Элемент 4

Чтобы добавить нумерованные списки, надо пункты просто пронумеровать. Пример
1. Первый Список
2. Второй Список
3. Третий Список
